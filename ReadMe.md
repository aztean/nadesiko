# 概要

　日本語プログラミング言語「なでしこ」とは、日本語でプログラムを作ることが出来るプログラミング言語です。

なでしこは以下の３点に特化した言語です。

- プログラミング言語の入門用に
- 事務作業のためのマクロ（定型作業に特化）
- プログラミングの楽しさを体験できるように


# インストール

　アーカイブの"SetupNako.exe"を実行してください。

　関連付けやショートカットなど開発に必要なツールが一通りインストールされます。
　また、バージョンアップしたときもインストール情報を調べて新しいファイルで上書きしてくれるので便利です。


# アンインストール

コントロールパネルのプログラムの追加と削除でなでしこをアンインストールしてください。


#【梱包ファイル】

-ReadMe.txt    ...  簡易説明書このファイル
-SetupNako.exe ...  簡単インストール用プログラム
-History.txt   ...  履歴ファイル
-licence.txt   ...  ライセンスに関する情報
-vnako.exe     ...  リッチGUI版なでしこ（ウィンドウ版）
-gnako.exe     ...  GUI版なでしこ（ウィンドウ版）
-cnako.exe     ...  コンソール版なでしこ（32bitコンソール）
-dnako.dll     ...  なでしこインタプリタDLL
-<doc>         ...  マニュアルフォルダ
-<tools>       ...  便利なツールライブラリを収めたフォルダ
-<sample>      ...  サンプルが入っているフォルダ
-<lib>         ...  なでしこライブラリが入っているフォルダ(*.nako)
-<plug-ins>    ...  ファイル/文字列処理のプラグインDLLがあるフォルダ
